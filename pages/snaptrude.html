Snaptrude

Freelance | React

Overview

Snaptrude is a cloud based concept design tool from scratch, helping architects to focus on the art of architecture by closing gaps present in the current design process as well as empowering the users with BIM (Building Information Modelling).

I worked as a freelancer in the team, building the dashboard and the plans page.

Tech Stack

React was used as a UI library to build the pages and the components along with styled-components to achieve consistency in the codebase.

Learnings

  • Gained a more stronger hold on creating CSS for small, individual elements.

  • Practiced code reviews for pull requests, to maintain code quality.

  • Explored various concepts like hooks & events for changing state.
